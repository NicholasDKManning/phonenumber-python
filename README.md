# ☎️ Phone Number Formatter  

A Python script that formats a 10-digit number into **(XXX) XXX-XXXX** format. Ensures valid input and applies string manipulation for correct formatting.  

🔹 Example:  
**Input:** `7137755733`  
**Output:** `(713) 775 - 5733`  

🚀 Run with:  
```bash
python phone_formatter.py
